Pixels to Proofs: Probabilistically-Safe Control in Latent World Models via Conformalized Robust MPC

SLS2 learns compact Markov latent world models from pixels and plans with conformalized System Level Synthesis to produce robust, safety-aware feedback control.

Paper PDF Code

Abstract

We present SLS2, a framework for safe feedback motion planning from pixels using robust model predictive control in learned latent world models. The approach trains an action-conditioned joint-embedding world model with compact Markovian latent states, enabling efficient gradient-based trajectory optimization through learned latent dynamics. To enforce safety despite imperfect latent predictions, SLS2 uses conformal prediction to calibrate latent error bounds and robust latent-space constraint sets, then plans with a GPU-accelerated System Level Synthesis robust MPC scheme. We further learn and conformalize latent constraint checkers so task constraints can be enforced during closed-loop execution.

Method

1

Learn Latent Dynamics

RGB observations are encoded into compact embeddings. Markov latent states combine the current embedding with finite-difference latent features, and an MLP predicts action-conditioned latent transitions.

2

Calibrate Uncertainty

Held-out rollouts calibrate one-step latent prediction errors, in-distribution latent ellipsoids, and task-specific latent safety classifiers with conformal prediction.

3

Plan with SLS

MPPI warm starts and GPU SLS robust MPC propagate conformal uncertainty sets into reachable tubes, tightening constraints over the planning horizon.
